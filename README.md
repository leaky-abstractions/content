# Leaky Abstractions — Videos

Source repository for the [Leaky Abstractions](https://youtube.com/@leakyabstractions) YouTube channel. Contains video scripts, animated scenes (built with [Motion Canvas](https://motioncanvas.io/)), and companion blog posts (built with [Eleventy](https://www.11ty.dev/)).

## What's in here

Each episode lives under `episodes/` and includes:

-   A video script, YouTube description, and captions
-   Animated scenes as TypeScript/TSX (Motion Canvas)
-   A companion blog post

Episodes can be standalone or grouped into a series under `episodes/_series/`.
